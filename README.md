### Install dependencies

```
[sudo] npm install -g webpack
npm i
```

### Build production

```
npm run build
```

### Run dev

Using webpack dev server

```
npm run dev
```
Open `http://localhost:3000/app/endscreen.html`

or

```
npm run no-server
```

macOS

open ./builds/MenuTube-darwin-x64/MenuTube.app
Run dev

npm run dev

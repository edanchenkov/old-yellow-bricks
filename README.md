### Install dependencies

```
[sudo] npm install -g webpack
npm i
```

### Build production

```
npm run build
```

then open `app/endscreen.html` or  `$ open app/endscreen.html` on mac os

### Run dev

Using webpack dev server

```
npm run dev
```
Open `http://localhost:3000/app/endscreen.html`

or

```
npm run no-server
```
